- 1.10.1.单引号字符串以及对引号转义

	- 与数一样,字符串也是值
	- 字符串有不同的表示形式(单引号,双引号,长字符串,原生字符串),但不同的表示形式都表示相同的字符串类型
	- 转义

		- 使用反斜杠(\)进行转义
		- 单引号字符串中双引号可以不用转义,直接表示
		- 双引号字符串中单引号可以不用转义,直接表示
		- 可使用长字符串,原始字符串,原始长字符串来避免转义

	- 拼接字符串:相加

	- 类str将值(包括字符串)转换为用户友好的字符串(转义)

	- 函数ascii将值转换为ASCII表示
	- 交互式解释器打印字符串的repr字符串

- 1.10.4.长字符串,原始字符串和字节

	- 在Python3中,所有字符串都是Unicode字符串
	- 常规字符串

		- 反斜杠对字符进行转义, 使能在字符串中包含原本(由于歧义或非字面字符而)无法包含的字符

	- 任何字符串表达式都不能以单个反斜杠结尾
	- (任何地方的)行尾的反斜杠会将行尾换行符转义,使其被忽略

		- 可用于将一行代码拆分为多行 (续行)

	- 1.10.4.1.长字符串

		- 表示

			- 使用三引号('''或""")表示

		- 机制

			- 其中(仅)单引号,双引号和换行符可以不用转义,直接表示.
			- 一切转义都生效

		- 用途

			- 表示跨越多行或同时含有单引号和双引号的字符串

	- 1.10.4.2.原始字符串

		- 表示

			- 使用前缀r或R表示

		- 机制

			- 不对反斜杠做特殊处理(不转义),将字符串包含的每个字符都视为普通字符

				- 单引号原始字符串中的单引号, 双引号原始字符串中的双引号, 原始字符串结尾的单个反斜杠需要转义, 但用于转义的反斜杠与被转义字符都将被视为普通字符而被包含在字符串中

		- 用途

			- 用于处理含有大量反斜杠的字符串,如DOS路径和正则表达式

						- 不要做过于严格的假设

				- 解码:bytes->str

					- str(<bytes>, encoding=<encoding>)

						- 更通用
						- 当不知道类似于bytes的对象属于哪个类时,更安全
						- 不要做过于严格的假设

				- 将文本写入文件或通过网络套接字发送

### 2.11. 1.11.小结

- 算法

	- 含义

		- 完成特定任务的方法

	- 编写计算机程序就是用计算机能够理解的语言描述一种算法
	- 程序主要由语句组成

- 表达式

	- 表达式是程序的一部分, 结果是一个值
	- 简单表达式是使用运算符和函数将字面值和变量组合起来得到的
	- 复杂表达式是通过组合简单表达式得到的

- 变量

	- 变量是指向值的名称
	- 通过赋值可将值赋给变量
	- 赋值是一种语句

- 语句

	- 语句是让计算机执行特定操作的指示

- 函数

	- 函数能接收参数并返回值

- 模块

	- 模块是文件

### 3.4. 2.2.通用序列操作

- 2.2.1.索引 (Index): <Sequence>[<Index>]

	- 通过元素编号访问单个元素
	- 每个元素对应一个 (正) 索引
	- 正索引从零开始顺序递增, 指出元素相对于序列首元素的偏移量
	- 负索引从零开始顺序递减, 指出元素相对于序列首元素 (而非尾元素) 的 (循环) 偏移量
	- 可以对序列字面量直接执行索引操作: <Sequence>[<Index>]
	- 若函数返回值是序列, 可以直接对函数返回值执行索引操作: <Function>(<Parameter>)[<Index>]

- 2.2.2.切片 (Slice): <Sequence>\[[<StartIndex>]:[<EndIndex>][:<Step>]\]

	- 访问特定范围的子序列
	- 切片返回的都是序列的副本
	- 抽取序列元素组成子序列

		- 步长为正

			- (均取正索引)
for (index = <StartIndex>; index < <EndIndex>; index += <Step>)
{
subSequence += [<Sequence>[index]]
}

		- 步长为负

			- (均取正索引)
for (index = <StartIndex>; index > <EndIndex>; index += <Step>)
{
subSequence += [<Sequence>[index]]
}

	- 索引

		- 可正可负
		- 起止索引不一定同正或同负
		- <StartIndex> 起始索引

			- 切片的起始索引
			- Include: 指定的元素包含在切片内
			- 默认

				- 步长为正

					- 序列首位置 (First Index, 0)

				- 步长为负

					- 序列末位置 (Last Index)

		- <EndIndex> 结束索引

			- 切片的结束索引
			- Exclude: 指定的元素不包含在切片内
			- 默认

				- 步长为正

					- 序列末位置 (Last Index) 后一个位置

				- 步长为负

					- 序列首位置 (First Index, 0) 前一个位置

		- 注意: <StartIndex> Include, <EndIndex> Exclude, 而非小索引 Include, 大索引 Exclude
		- "虚索引": <LastIndex> + 1, 最后一个元素的索引+1, 代表最后一个元素的后一个位置

	- <Step> 步长

		- 抽取元素的跨度
		- 可正可负
		- 不能为0, 否则无法移动
		- 默认为1: <Sequence>\[[<StartIndex>]:[<EndIndex>]\] 等价于 <Sequence>\[[<StartIndex>]:[<EndIndex>]:1\]
		- 若起止索引方向与步长方向相反, 则返回空序列

- 2.2.3.序列相加: <Sequence> + <Sequence>

	- 使用加法运算符拼接序列
	- 不能拼接不同类型的序列, 如列表和字符串

- 2.2.4.乘法: <Sequence> * <IntValue>

	- 将序列与数x相乘时, 将重复序列x次来创建一个新序列

- 初始化列表

	- 空列表: 使用不包含任何内容的一组方括号表示: []
	- None: 表示"什么都没有"的值
	- 将列表的长度初始化为n (创建一个可包含n个元素的列表, 但不在列表中添加任何内容): <Sequence> = [None] * n

- 2.2.5.成员资格

	- 布尔值

		- 真: True
		- 假: False

	- 布尔运算符: 检查是否满足指定条件, 并返回对应布尔值的运算符
	- 使用成员资格布尔运算符in检查指定对象是否是序列 (或其他集合) 的元素

		- <Element> in <Sequence>
		- <Element> in <Set>
		- 特别的, 对字符串, 使用布尔运算符in检查指定字符串是否是序列的子串

			- <Str> in <Str>

- 迭代 (Iteration)

	- 对每个元素都执行特定的操作

### 3.5. 2.3.列表: Python的主力

- 列表是可变的
- 2.3.1.类型 list

	- 字符串与列表

		- 与列表不同, 字符串不可修改
		- list(<Str>): 使用字符串创建列表
		- ''.join(<StrList>): 将字符串列表拼接为字符串

- 2.3.2.基本的列表操作

	- 列表是序列的子类: 可对列表执行任何序列操作
	- 给元素赋值: <Sequence>[<Index>] = <Value>

		- 使用索引引用元素
		- 使用赋值语句赋值
		- 注意: 不能给不存在的元素赋值, 必须先初始化列表, 使其长度大于指定索引

	- 删除元素与子列表

		- 使用del语句从列表中删除元素: del <List>[<Index>]
		- 使用del语句从列表中删除子列表: del <List>[[<StartIndex>]:[<EndIndex>][:<Step>]]
		- 删除后, 元素或子列表消失, 列表长度缩短
		- del语句还可用于删除其他东西, 如字典, 变量等

	- 给切片赋值: <List>[[<StartIndex>]:[<EndIndex>][:<Step>]] = <List>

		- 可用于同时对多个元素赋值
		- 当切片步长为1时

			- 赋值相当于用列表替换子列表
			- 用于赋值的列表与被赋值的子列表长度不必一致

				- 通过对空切片赋值, 可在不修改原有元素的情况下插入列表
				- 通过对切片赋空列表值, 可在不修改其余元素的情况下删除切片

		- 当切片步长非1时

			- 对切片子列表所指定的元素依次对应赋值
			- 用于赋值的列表与被赋值的子列表长度必须一致

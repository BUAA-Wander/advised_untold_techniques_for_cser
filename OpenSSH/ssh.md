# ssh

- `ssh [OPTIONS] [ssh://][USER@]HOST[:PORT] [COMMAND]`
    - `-v[v[v]]`: Verbose
    - `-4`: IPv4 Only
    - `-6`: IPv6 Only
    - `-o "OPTION VALUE"`: Overriding Option
    - `-F CONFIG_PATH`: Overriding Config
    - `-G`: Eval Config
    - `-J [ssh://][USER@]HOST[:PORT]`: Jump
    - `-i IDPATH`: Private Key
    - `-l USER`: User Name
    - `-p PORT`: Port
    - `-V`: Version
